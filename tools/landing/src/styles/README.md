# Landing — styles

Global CSS for the landing page.

| File | Purpose |
|---|---|
| `global.css` | Color tokens (the five-color palette + graphite for wireframes), type ramp, spacing scale, reset, and shared utility classes (`.btn`, `.terminal`, `.xref`, `.wordmark`). |

All section-specific styles live as scoped `<style>` blocks inside each component in `../components/`. Global tokens here define what every component reaches for.

Brand source of truth: [`branding/palette.md`](https://github.com/specscore/marketing/blob/main/branding/palette.md) in the marketing repo. Hex values here mirror that doc; if they diverge, the marketing repo wins.
